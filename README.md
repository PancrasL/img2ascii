# Introduction
A small tool for converting images to ASCII characters.

# QuickStart
## Install Dependencies
```bash
python install -r requirements
```

## Run
```bash
# Start By Shell
python img2ascii.py

# Start By GUI
python gui.py
```

## Package as executable program
```bash
pip install pyinstaller
pyinstaller gui.py -F
```
